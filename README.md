<h1>Api nodeJS express utilizando o Banco de Dado NEO4J</h1>


<h4>Pacotes Necessarios</h4>
<pre>
npm install neo4j-driver 
npm install nanoid 
npm install dotenv
npm install express
</pre>

<h4>Configuração do Dotenv</h4>
<pre>
.env
PORT=3000
#Database
url=
db_username=
db_password=
database=</br>
</pre>
